# 忆流Web知识记住辅助软件

## 项目简介

忆流Web知识记住辅助软件是一款基于Web的知识点管理系统，专为帮助用户高效记忆和管理各类知识点而设计。该软件支持知识点的添加、编辑、导入导出、学习统计等功能，适合学生、教师、专业人士等各类用户使用。

## 功能特点

- **知识点管理**：支持添加、编辑、删除知识点，包含文本、拼音、释义、例句等多种字段
- **学习模式**：提供智能学习模式，根据优先级自动调整学习顺序
- **统计分析**：可视化展示学习情况，包括优先级分布、记忆水平统计等
- **导入导出**：支持Excel格式的知识点导入导出，方便批量管理
- **收藏功能**：支持知识点收藏和收藏原因记录
- **公式支持**：内置MathJax支持，可渲染LaTeX数学公式
- **夜间模式**：支持切换夜间模式，保护视力

## 技术栈

- 前端：HTML5 + CSS3 + JavaScript
- 图表库：Chart.js
- 数学公式：MathJax 3.2.0
- Excel处理：SheetJS (xlsx.full.min.js)

## 使用说明

### 知识点管理
1. 点击「添加」按钮添加新知识点
2. 使用「导入」功能批量导入知识点
3. 在列表中可进行编辑、删除、重置等操作

### 学习模式
1. 点击「学习」按钮进入学习模式
2. 查看知识点，点击「显示释义」查看详细内容
3. 根据记忆情况选择「认识」、「模糊」或「不认识」
4. 系统会根据反馈调整知识点优先级

### 统计分析
1. 点击「统计」按钮查看学习统计
2. 查看各类图表和统计数据
3. 根据统计结果调整学习策略
